== Description ==

A minimal WordPress theme designed specifically for maximum compatibility with Brizy page builder. This theme is NOT built by the official Brizy team. It provides only essential WordPress functionality while letting Brizy handle all design and layout decisions.

== Installation ==

1. Upload the `brizy-theme` folder to the `/wp-content/themes/` directory.
2. Activate the theme through the 'Appearance' menu in WordPress.
3. Install and activate the Brizy page builder plugin from the WordPress Plugin Directory.
4. Create a new page and select the "Brizy Template" to start building with Brizy.
5. Start building your pages with Brizy!

== Frequently Asked Questions ==

= Do I need to use Brizy with this theme? =
While this theme is optimized for Brizy, you can use the standard WordPress block editor or other page builders. However, the theme is specifically designed to work best with Brizy.

= Will this theme work on its own without Brizy? =
Yes! This theme includes all essential WordPress functionality. It can work standalone, though it's best utilized with Brizy page builder.

= How do I create a Brizy page? =
Create a new page or post, then select "Brizy Template" from the page template dropdown. Open the page editor to use Brizy's visual builder.

= Is this theme officially made by the Brizy team? =
No. This theme is made by REVENTOR and is not affiliated with the official Brizy team. It's designed to provide maximum compatibility with Brizy.

= Can I customize the theme without using Brizy? =
Yes. You can use the WordPress Customizer to add a custom logo and manage your site title. The theme uses minimal CSS to avoid conflicts with your custom designs.
